## 5.1 Generate 3 Design Directions

Create 3 genuinely different visual interpretations using the same foundation.

Each direction must vary in:
- **Layout composition** — asymmetric vs grid vs editorial vs dense
- **Visual weight** — light/airy vs bold/heavy vs balanced
- **Personality emphasis** — which aspect of the product's personality is amplified

```
**Direction 1: [Name]**
- Layout: [description]
- Visual weight: [description]
- Personality: [what it emphasizes]
- Best for: [when this direction shines]

**Direction 2: [Name]**
- Layout: [description]
- Visual weight: [description]
- Personality: [what it emphasizes]
- Best for: [when this direction shines]

**Direction 3: [Name]**
- Layout: [description]
- Visual weight: [description]
- Personality: [what it emphasizes]
- Best for: [when this direction shines]
```

## 5.2 Evaluate

| Criteria | Dir. 1 | Dir. 2 | Dir. 3 |
|----------|--------|--------|--------|
| Impact visuel | ★★★ | ★★ | ★★★ |
| Cohérence avec le tone | ★★★ | ★★★ | ★★ |
| Lisibilité / UX | ★★ | ★★★ | ★★ |
| Originalité | ★★ | ★★ | ★★★ |
| Faisabilité technique | ★★★ | ★★ | ★★ |

## 5.3 User Choice

> "Voici 3 directions visuelles basées sur ta fondation. Chacune raconte une histoire différente avec les mêmes ingrédients.
>
> Laquelle te parle le plus ? Tu peux aussi mixer : 'le layout de la 1 avec la densité de la 3'."

Wait for user choice. Allow mixing elements across directions.

## 5.4 Refine Chosen Direction

Once chosen, detail the direction with specifics:

- Exact spacing for key elements
- Component hierarchy (what stands out, what recedes)
- Motion strategy (what animates, what's static)
- Interactive states (hover, focus, active, disabled)
- Empty states and loading states
